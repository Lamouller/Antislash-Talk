💻 COMMANDES ESSENTIELLES

  # Développement
  pnpm dev              → Web app (dev mode, rapide)
  pnpm deploy:local     → Stack complète (12 services)

  # Docker
  pnpm docker:up        → Démarrer services
  pnpm docker:down      → Arrêter services
  pnpm docker:logs      → Voir logs
  pnpm docker:rebuild   → Rebuild & restart

  # Database
  pnpm db:migrate       → Appliquer migrations
  pnpm db:types         → Générer types TS
  
  # Build
  pnpm build            → Build production
  pnpm typecheck        → Vérifier types
  pnpm lint             → Vérifier code

═══════════════════════════════════════════════════════════════

⚙️ CONFIGURATION

  1. Copier : cp env.monorepo.example .env.monorepo
  2. Éditer : nano .env.monorepo
  3. Modifier :
     - POSTGRES_PASSWORD (mot de passe DB)
     - JWT_SECRET (clé JWT, min 32 chars)
     - SITE_URL (pour production)
